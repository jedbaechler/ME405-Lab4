Detailed Description

Reads serial inputs and puts 2 columns in lists.

This file decodes the csv data written through the serial port and instantiates them as two lists. These two lists are converted to floats, where each row is then appended. With these two lists we are able to instantly print the step response. This program is adapted from our plotter for lab 2.
